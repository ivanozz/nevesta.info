Краткое описание:
==================================

/core - ядро с вспомогательными классами и константами

/data - исходные данные

/application - приложение

/application/controller/MainController - базовый контроллер

подключает вспомогательные классы,
осуществляет роутинг приложения

/application/controller/ImportController - контроллер для наполнения базы данных

обрабатывает входные данные,
заполняет ими базу данных

/application/controller/IndexController - основной контроллер

выводит листинг фотографий, постраничную навигацию, панель фильтрации,
панели с тегами, выставляет лайки к фотографиям

/application/model - контроллеры оперируют данными с помощью соответствующих моделей

/application/view - содержит шаблоны приложения и ресурсы для их отображения
